# YourDOST Presentation

Founder's Office Assignment presentation for YourDOST - a mental health support platform for organizations.

## Overview

This is an interactive HTML presentation showcasing:
- Q1 target enterprises and outreach strategy
- Outreach message templates
- Swiggy proposal and upsell strategy
- Roundtable to revenue conversion plan

## Features

- 5-slide interactive presentation
- Navigation via buttons, keyboard arrows, or dot indicators
- Responsive design with YourDOST branding
- Animated transitions between slides

## How to View

Simply open `yourdost_presentation.html` in any modern web browser. No build process or dependencies required.

## Deployment

This project can be deployed to:
- GitHub Pages
- Netlify
- Vercel
- Any static hosting service

## Tech Stack

- Pure HTML, CSS, and JavaScript
- No external dependencies
- Google Fonts (Nunito, DM Sans) loaded via CDN
